# Apexcify-Technologys
## 🔹 CGPA-Calculator.cpp

This project is a CGPA Calculator developed in C++. It allows users to input course details such as credit hours and grades, and calculates the GPA for the current semester as well as the overall CGPA.

The program first asks the user to enter the number of courses. It then uses a loop to take input for each course, including credit hours and grades. Based on the entered grade (such as A, B+, C, etc.), the program uses conditional statements to convert grades into their corresponding grade points.

After collecting all inputs, the program calculates the total grade points by multiplying grade points with credit hours, and also keeps track of total credit hours. The GPA is then calculated by dividing total grade points by total credit hours.

In addition, the program asks for the previous CGPA and the number of semesters completed. Using this information, it calculates the updated overall CGPA.

This project demonstrates the use of basic C++ concepts such as loops, conditional statements, user input/output, and arithmetic operations. It is useful for understanding how academic performance calculations like GPA and CGPA can be automated using programming.
