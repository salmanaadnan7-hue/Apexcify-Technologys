# Apexcify-Technologys

This repository contains multiple **C++ console-based projects** developed as part of internship tasks. Each project demonstrates core programming concepts such as object-oriented programming (OOP), file handling, data structures, and problem-solving techniques.

---

## 📌 Projects Included

### 1. 🎓 CGPA Calculator

A simple program to calculate GPA and overall CGPA.

**Features:**

* Takes number of courses as input
* Accepts credit hours and grades
* Calculates GPA for the semester
* Computes updated CGPA using previous record

**Concepts Used:**

* Loops & conditionals
* String handling
* Basic mathematical calculations

---

### 2. 🔐 User Management System (Login/Register)

A file-based authentication system.

**Features:**

* User registration with username & password
* Login verification
* Stores credentials in `.txt` files
* Prevents duplicate usernames

**Concepts Used:**

* File handling (`ifstream`, `ofstream`)
* Classes & objects
* Input validation

---

### 3. 🧩 Sudoku Solver

A program that solves a 9x9 Sudoku puzzle using backtracking.

**Features:**

* Accepts incomplete Sudoku grid (0 for empty cells)
* Solves using recursion
* Displays completed grid

**Concepts Used:**

* Recursion
* Backtracking algorithm
* 2D arrays

---

### 4. 🏦 Banking Management System

A mini banking system simulation using OOP.

**Features:**

* Customer creation
* Multiple account management
* Deposit & withdrawal operations
* Fund transfer between accounts
* Transaction history tracking
* Display account details

**Concepts Used:**

* Object-Oriented Programming (Classes & Objects)
* Vectors (dynamic arrays)
* Encapsulation
* Transaction tracking

---

## 🛠️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/Apexcify-Technologys.git
   ```

2. Open any `.cpp` file in a C++ compiler (e.g., Dev C++, CodeBlocks, VS Code)

3. Compile and run:

   ```bash
   g++ filename.cpp -o output
   ./output
   ```
